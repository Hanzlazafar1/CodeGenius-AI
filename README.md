# CodeGenius AI

Full-stack multi-agent software engineering assistant.

## Quick Start

### 1. Backend
```bash
cd backend
pip install -r requirements.txt
# Edit .env and add your GROQ_API_KEY
uvicorn main:app --reload --port 8000
```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```

App runs at http://localhost:5173

- ⚡ Algorithm Generator
- 🔍 Code Explainer
- 🐛 Bug Analyzer
- 📋 SRS Extractor
